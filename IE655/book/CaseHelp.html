Which browser to use and how to open a_StartCase.html ?

We strongly recommend that you use the latest browser. (Note: several different browsers are available free from the Internet).

Again, try to use the latest browser. This program will run on all browsers available at the time of writing in the USA.

a_StartCase.html is the main contents, or "home" html file for this program. In other words this is the file everything is object linked to. So it is important that you open this file from the browser first, nevertheless you can open the program from any .html file in the CD directory. You'll eventually get to this main title file (unless you open a media file, or similar that is linked via the "Back" button in the browser).

This is how to open a_StartCase.html from Netscape running on a PC. (Note that the IE (Internet Explorer) sequence is very similar with minor terminology differences. The same applies to Mac users, although it is even simpler on Apple Macs).

Step 0. Make sure that you have this CD-ROM, or DVD-ROM in your CD / DVD drive! (Trust us, there are some customers, NOT you, who expect to run this program without the CD / DVD in the drive...there are always some challenges for all of us...)

Step 1. Execute the browser the usual way (double click on it). If you are on-line the Internet, the browser will run without any complaints. If you are off-line the Internet your browser will complain about not having been able to establish a network pipe (or connection). Accept this and click on OK.

Step 2. In the File menu, select 'Open Page', then 'Choose File', and having opened the 'Resources' folder (or directory), select 'a_StartCase.html' in the appropriate CD / DVD drive ('look in') in your machine. (Files are sorted in most cases in an alphabetic order). Note, that some PCs won't show the .html extension after 'a_StartCase'. This is fine; no problem, it is the same file.

Step 3. Then double click on 'a_StartCase.html' and click on 'Open' (in navigator mode). This will open the a_StarCase file, the home and the title file of this program. Once you see our opening screen, just click on 'Contents' to start the show...or in other words, to 'open this electronic book'.

Enjoy and explore. There are several Internet links in this program which you can execute only if you are on-line, nevertheless the rest of the program behaves exactly the same way wether or not you are on-line or off-line.

If you are off-line and click on an on-line Internet link you'll get a simple error message from the browser that will not cause any harm.

How to play the digital videos?

All digital videos in this program are stored in Apple Computer Inc.'s international standard QuickTime format. (More on QuickTime at the http://www.apple.com website).

This is a very robust, high quality and well established cross platform format. If you have difficulties playing the videos please make sure that you have at least QuickTime version 4.0 or later installed on your machine, and that you can access it through your browser.

QuickTime is very user friendly and if you install it as "full" it will automatically integrate itself into your browser so that when you click on a movie icon in this program it will automatically start to play in most browsers. (You can also open these files in a separate browser window. This way you can position the media files wherever you want them on your screen. Note again, that this behavior will vary depending on your browser).

Please feel free to explore the interactive start/ stop, frame-by-frame, etc. features of QuickTime. It is a great program!

Also note that the same QuickTime ver. 4.0 or later program executes the interactive 3DVR as well as the panoramic images. This means that if you have installed it properly it will do all media jobs for you! (It will even be able to open your .jpg image files).

How to view 3D virtual reality (3DVR) object movies?

3DVR objects are basically sophisticated multiple digital videos that are controlled by you the user! To play them you need Apple Computer Inc.'s QuickTime ver 4.0 or later program. More on QuickTime at the http://www.apple.com website).

You can activate this feature by clicking onto the movie's poster frame and then moving the mouse up and down whilst pressing the mouse button down. (If there is no change, it means that that particular 3DVR object is not animated... sorry... sometimes we have to sleep too...)

What can I do if I have a slow computer?

Obviously 'slow' can mean different speeds, nevertheless if you have a relatively old machine, such as an 'oldish' 300 MHz PC or PowerMac you will still be able to enjoy the show, but your videos might be slow, the images might look grainy and the 3DVR objects might be a bit jerky.

It is important to recognize that despite what the great public believes, and some salesmen tell you, the CPU clock speed is not the only important parameter of your machine.

The CD drive speed (typically between 12x and 24x, or even faster, meaning 12 x 150 kb/sec to 24 x 150 kb/sec transfer rates), your bus clock speed, your hard disk access time and transfer rates and the way your computer is integrated are very important parameters too. This is why you can experience significantly different overall performance in different PCs even if the clock speed is the same. (Not only that, but the specified CD drive speeds, e.g. 24x mean peak performance speeds, NOT average speeds...interesting marketing... agreed?)

Besides all this, there are major architectural design issues that effect the computer's 'speed'.

As a consequence of all this, at the user end, if you don't see a 3DVR object or one of the longer videos loading this is most likely because you are low on memory.

This program will run on as low as 32 Mbytes, with the operating system and the browser together (!) nevertheless the more memory you have and the more video RAM you have the faster you will fly... (As they say in this business 'the bigger and faster, the better...')

Also note, that your CD-ROM / DVD-ROM might go to sleep, then wake up and eventually run, or just spin slowly (at 8x or so). There is not much we or you can do about it. If this is the case you might want to consider replacing your drive or even your machine soon.

Is it normal that the screen layout of the program looks a little bit different on different computers?

Yes! This is due to the fact that we have allowed maximum flexibility in terms of selecting our font types and sizes, as well as because the actual layout you see on your screen depends on the screen resolution and color setting of your computer, as well as the way you have configured your browser. Many components... we know! Imagine, there are millions of computers out there, all configured in a different way... running under different browsers at different speeds, etc. But this is freedom. So it is OK. We support open source, therefore you can change, adjust and even edit and modify...

If you are unhappy with what you see, change (if possible) the resolution of your screen to at least above 800 x 600, around 1024 x 768 (or whatever best is offered; different on CRT and LCD screens), the number of colors to 32 bit (millions of colors), but at least 16 bit, and the browser font type to Arial 12 or Helvetica 12. This will give you the best possible view and layout design.

Note that some of the 3DVR objects in this program might be visually large, therefore you should view them in 1024 x 768 pixel or higher resolution to see the full image on the screen. If you cannot set your PC to comply with this relatively high resolution and therefore don't see the full 3DVR image, this is not a major problem because by clicking on it and moving the mouse left and right it will still work well.

If you want to fly, use a reasonable machine, preferably around 1GHz, considered 'reasonable' at the time of writing, nevertheless as we have stated earlier, you'll be able to run our programs well around 300-500 MHz too...

Again: in the IT industry it is true, that the "faster and larger the better".... Have fun!
